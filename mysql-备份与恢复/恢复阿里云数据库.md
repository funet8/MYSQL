请确保您的Percona XtraBackup版本正确：

# 通过binlog日志恢复数据

1.下载binlog日志
```
wget -c 'URL路径' -O /data/tmp/binlog/mysql-bin1
```

2.导入数据库
```
mysqlbinlog  -d 指定数据库
mysql -f 忽略报错，强制导入
mysqlbinlog  -d '指定数据库'  /data/tmp/binlog/mysql-bin1 | mysql -u root -S /home/data/mysql/61921/mysql61921.sock -f
```
